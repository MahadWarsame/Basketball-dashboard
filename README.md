# Basketball Dashboard

A fullstack basketball statistics dashboard web-app built with React, Tailwind CSS, and D3.js. Live NBA data is pulled from an API to display dynamic charts and analytics.

## Live Demo

Check out the live project here: https://basketball-dashboard-gamma.vercel.app

## Features

- Interactive charts and graphs with D3.js and Nivo  
- Tailwind CSS for responsive and modern design  
- React for fast and dynamic UI updates  
- Live NBA data fetched from API  

## Technologies Used

React | Tailwind CSS | D3.js | Nivo | JavaScript | HTML | CSS
